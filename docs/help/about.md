# Sobre o SwitchPilot

**Versão**: v1.7.1  
**Desenvolvedor**: Fabiano Brandão

O **SwitchPilot** é uma ferramenta de automação para transmissões ao vivo que monitora o conteúdo da tela (Região PGM) e executa ações automáticas no **OBS Studio** ou **vMix** quando cenas específicas são detectadas.

---

## 🚀 Funcionalidades Principais
*   **Detecção Visual Inteligente**: Usa algoritmos de visão computacional (Histograma + NCC + LBP) para reconhecer cenas.
*   **Integração Nativa**: Controla OBS via WebSocket e vMix via API HTTP.
*   **Leve e Rápido**: Otimizado para consumir poucos recursos durante a live.
*   **Multifonte**: Captura monitores inteiros, janelas específicas ou feeds NDI.
